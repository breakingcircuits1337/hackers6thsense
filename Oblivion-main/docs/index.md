# Oblivion Red-Team

Oblivion is an LLM-assisted adversary emulation and red-team automation framework for blue-team labs, instructors, and security researchers.

## Features

- Interactive CLI and browser dashboards
- Dynamic attack planning powered by LLMs
- Modular attack modules (scan, exploit, phishing, exfiltration, etc.)
- Scenario runner for multi-step kill chains
- REST API & WebSockets for automation and real-time monitoring
- Advanced safety controls: policy.yaml, dry-run, per-technique allow lists
- Containerized and pip-installable
